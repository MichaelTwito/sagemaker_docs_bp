Registering Module Documentation

The registering module handles model registration in the SageMaker Model Registry.

Class: Register

Constructor

def __init__(self, pipeline_session, role, pipeline_name, base_job_name, s3_base_path, evaluator, model):

Parameters:

  • pipeline_session: SageMaker pipeline session
  • role: IAM role for execution
  • pipeline_name: Name of the pipeline
  • base_job_name: Base name for jobs
  • s3_base_path: S3 path for data storage
  • evaluator: Evaluator instance
  • model: Model instance

Key Methods

create_registery()

Creates a model package in the Model Registry.

create_step()

Creates a ModelStep for the pipeline.

Model Registry Configuration

Default configuration:

  • Content Types: text/csv
  • Response Types: text/csv
  • Inference Instances: ml.t2.medium
  • Transform Instances: ml.m5.large
  • Default Approval Status: PendingManualApproval
