Modeling Module Documentation

The modeling module handles model creation and configuration in the SageMaker pipeline.

Class: Model

Parameters:

  • pipeline_session: SageMaker pipeline session
  • role: IAM role for execution
  • pipeline_name: Name of the pipeline
  • base_job_name: Base name for jobs
  • s3_base_path: S3 path for data storage
  • bucket_name: S3 bucket name
  • bucket_prefix: S3 prefix for model artifacts
