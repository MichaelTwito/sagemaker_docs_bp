Modeling Module Documentation

The modeling module handles model creation and configuration in the SageMaker pipeline.

Class: Model

Initializes the model creation module, setting up parameters for defining the model pipeline.

Parameters:

  • pipeline_session: SageMaker pipeline session
  • role: IAM role for execution
  • pipeline_name: Name of the pipeline
  • base_job_name: Base name for jobs
  • s3_base_path: S3 path for data storage
  • bucket_name: S3 bucket name
  • bucket_prefix: S3 prefix for model artifacts

How the Model Creation Works

The model creation process follows these steps:

  1. Retrieving Trained Model:
    • The trained XGBoost model is retrieved from the SageMaker training step.
  2. Defining the Model:
    • A PipelineModel object is created using the trained model.
    • Network configuration (VPC, security groups) is defined for secure execution.
  3. Saving the Model:
    • The pipeline model is stored for deployment.
